🚀 Project Overview

This project analyzes airline flight delay data to uncover key patterns, causes, and trends affecting flight performance. The goal is to derive meaningful insights using SQL, Python (Pandas), and Power BI to support data-driven decision-making in airline operations.

🛠️ Tech Stack
SQL – Data extraction, transformation, and querying

Python (Pandas, NumPy) – Data cleaning, analysis, and feature engineering

Power BI – Data visualization and interactive dashboard

Excel / CSV Dataset – Source data

📂 Dataset
Airline Flight Delay Dataset (sourced from Kaggle)

Includes features like: airline, origin, destination, departure delay, arrival delay, cancellation, delay reasons (weather, carrier, etc.)

🔍 Key Analysis Performed

📌 SQL Analysis

Identified most delayed routes (origin-destination pairs)

Calculated average delay per airline

Analyzed total flights per airport

Breakdown of delays by cause (weather, carrier, security, etc.)

Monthly average departure & arrival delays

Top 5 airports with highest delayed departures

Cancellation rate per airline

🐍 Python (Pandas) Analysis

Cleaned and handled missing/invalid delay values

Analyzed delay patterns by day of the week

Studied weather impact on flight delays

Calculated on-time performance rate

Identified delay trends by time of day

Explored seasonal trends (monthly/quarterly)

(Optional) Analyzed aircraft turnaround time


📊 Power BI Dashboard
📈 Delay trends over time (daily/monthly)

📊 Average delays per airline (bar chart)

🥧 On-time vs delayed flights (pie chart)

🌍 Airport-wise delay analysis using geo maps

🔥 Heatmap of delays by weekday & hour

📌 KPI Cards: Avg Delay, Total Flights, Cancellation Rate

🎯 Interactive filters (airline, airport, delay cause, month)


📈 Key Insights

Identified peak delay periods and high-risk routes

Weather and carrier issues are major contributors to delays

Certain airports consistently show higher delay frequency

Seasonal trends impact overall flight performance


🎯 Outcome

This project demonstrates strong skills in:

Data extraction & querying (SQL)

Data cleaning & analysis (Python)

Data visualization & storytelling (Power BI)
